<h3 align="center">TaskFlow 📋<br><br>Uma aplicação de lista de tarefas (to-do list) simples, moderna e eficiente, desenvolvida para demonstrar a integração entre Next.js (React) no frontend e Spring Boot no backend com Java 21+, banco de dados PostgreSQL e todo o ambiente containerizado com Docker. O projeto também inclui pipelines de CI/CD com GitHub Actions.</h3>

###

<h4 align="left">📌 Problema que resolve<br><br>No dia a dia, precisamos de uma ferramenta rápida e intuitiva para organizar nossas tarefas. O TaskFlow resolve exatamente isso: permite criar, editar, marcar como concluída e excluir tarefas de forma simples, com uma interface limpa e responsiva. É a solução ideal para quem quer manter o foco e a produtividade.<br><br>✨ Funcionalidades<br><br>✅ Criar novas tarefas<br>📝 Editar tarefas existentes (com duplo clique no texto)<br>✔️ Marcar/desmarcar tarefas como concluídas<br>🗑️ Excluir tarefas<br>📱 Design responsivo (funciona em desktop e mobile)<br>🎨 Interface moderna com Tailwind CSS e componentes shadcn/ui</h4>

###

<h4 align="left">🛠️ Tecnologias utilizadas<br><br>Frontend<br><br>- Next.js 14+ (React framework)<br>- TypeScript<br>- Tailwind CSS para estilização<br>- shadcn/ui para componentes acessíveis e customizáveis<br>- Jest e Testing Library para testes<br><br>Backend<br><br>- Spring Boot 3.2+ (Java 21)<br>- Spring Data JPA para persistência<br>- PostgreSQL como banco de dados<br>- Maven para gerenciamento de dependências<br>- JUnit 5 e Mockito para testes<br>- Infraestrutura e DevOps<br>- Docker e Docker Compose para containerização<br>- GitHub Actions para CI/CD (integração contínua e deploy automatizado)<br>- Repositório único, mas com frontend e backend em diretórios separados</h4>

###

<h4 align="left">📋 Pré-requisitos<br><br>- Para rodar o projeto localmente, você precisará ter instalado:<br>- Docker e Docker Compose (recomendado)<br>- Ou, se preferir executar manualmente:<br>- Java 21+ e Maven (para o backend)<br>- Node.js 18+ e npm/yarn (para o frontend)<br>- PostgreSQL (pode ser via Docker)</h4>

###

<h4 align="left">🚀 Como executar<br><br>- Com Docker Compose (recomendado)<br>- Na raiz do repositório, execute:<br>- bash<br>- docker-compose up --build<br>- A aplicação estará disponível em:<br>- Frontend: http://localhost:300<br>- Backend API: http://localhost:8080/api<br>- Banco de dados PostgreSQL: localhost:5432 (usuário admin, senha admin123, database todo_db)</h4>
